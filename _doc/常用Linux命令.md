

# 通用

## 💻查看系统发行版信息

发行版信息指明了你使用的是哪一款Linux发行版（如Ubuntu、CentOS）及其版本号。

| **命令或文件**        | **说明**                                                     | **适用系统/特点**                                            |
| --------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| `cat /etc/os-release` | 显示系统标识信息，包含`NAME`, `VERSION`, `PRETTY_NAME`等关键字段。 | **最推荐**，现代Linux发行版通用。                            |
| `cat /etc/*-release`  | 读取`/etc`目录下所有以`-release`结尾的文件内容，信息全面。   | 兼容性最强，几乎所有Linux系统都支持。                        |
| `hostnamectl`         | 除了主机名，还会显示操作系统和内核版本，信息集成度高。       | 仅适用于使用`systemd`作为初始化系统的发行版。                |
| `lsb_release -a`      | 显示符合Linux标准规范（LSB）的详细信息，包括发行商、版本号和代号。 | 主流发行版（如Ubuntu、Debian），需要系统已安装`lsb-release`包。 |
